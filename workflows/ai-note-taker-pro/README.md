# AI Note Taker Pro

Self-contained n8n package.

## Files
- `workflow.json`
- `.env.example`

## Input contract
- audio_url, workspace_id

## Output contract
- structured notes, actions, owners, publish payload

## Quick start
1. Import `workflow.json` into n8n.
2. Configure credentials using env keys in `.env.example`.
3. Run with sample input from first Set node.
