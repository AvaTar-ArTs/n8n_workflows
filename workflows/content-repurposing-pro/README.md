# Content Repurposing Pro

Self-contained n8n package.

## Files
- `workflow.json`
- `.env.example`

## Input contract
- source_content, brand_voice, channels

## Output contract
- channel-specific assets (LinkedIn, TikTok, YouTube, X, newsletter)

## Quick start
1. Import `workflow.json` into n8n.
2. Configure credentials using env keys in `.env.example`.
3. Run with sample input from first Set node.
