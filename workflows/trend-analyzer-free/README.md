# Trend Analyzer Free

Self-contained n8n package.

## Files
- `workflow.json`
- `.env.example`

## Input contract
- keyword, variant_1, variant_2, intent

## Output contract
- single keyword SEO/AEO analysis + upgrade CTA

## Quick start
1. Import `workflow.json` into n8n.
2. Configure credentials using env keys in `.env.example`.
3. Run with sample input from first Set node.
